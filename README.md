###Iris Flower Classification

CodSoft Data Science Internship – Task 3
📌 Project Overview

The Iris Flower Classification project is a Machine Learning project that classifies iris flowers into three different species:

Setosa
Versicolor
Virginica

The classification is based on flower measurements such as sepal length, sepal width, petal length, and petal width.

🎯 Objective

The main objective of this project is to:

Build a Machine Learning classification model
Train the model using the Iris dataset
Predict the species of iris flowers accurately
Understand classification algorithms in Data Science
📂 Dataset Information

The Iris dataset contains the following features:

Sepal Length
Sepal Width
Petal Length
Petal Width
Species

Total Classes:

Iris-setosa
Iris-versicolor
Iris-virginica
    
🛠 Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
    
🤖 Machine Learning Algorithm
Logistic Regression / Decision Tree / K-Nearest Neighbors (KNN)
    
📊 Steps Performed
Imported required libraries
Loaded the Iris dataset
Performed data preprocessing
Visualized dataset using graphs
Split dataset into training and testing data
Trained Machine Learning model
Predicted flower species
Evaluated model accuracy
    
📈 Model Evaluation

The model performance was evaluated using:

Accuracy Score
Confusion Matrix
Classification Report

✅ Conclusion

This project demonstrates how Machine Learning algorithms can classify flowers based on their physical measurements. It is a beginner-friendly Data Science project that helps understand classification techniques, model training, and prediction.
